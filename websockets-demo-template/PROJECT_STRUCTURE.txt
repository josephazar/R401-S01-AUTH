DÉMARRAGE RAPIDE
================
1. cd backend && npm install
2. cd ../frontend && npm install
3. Démarrer MongoDB: mongod
4. Terminal 1: cd backend && npm run dev
5. Terminal 2: cd frontend && npm run dev
6. Ouvrir http://localhost:5173
